# 7thShop

Buyer-ready, cross-play economy shop plugin for Paper/Spigot servers. Built for 7th Heaven, designed to feel premium to players and simple to run for staff.

**What It Does**
7thShop provides a full in-game shop, selling, and marketplace experience with dynamic pricing, automated selling, and a second currency system.

**Core Features**
- Player-facing shop GUI with categories, search, and clean navigation
- Instant sell menu and sell-inventory tools
- Dynamic pricing with stock curves, negative stock inflation, and time-based price growth
- AutoSell chests with limits, setup flow, refunds, and tax options
- Player shop listings and browsing
- Gems secondary currency with boosters, leaderboards, and admin tools
- Item worth display modes for players
- Transaction logging and configurable sounds

**Systems & Integrations**
- Vault economy support
- ProtocolLib for advanced GUI behavior
- PlaceholderAPI placeholders (optional)
- ItemsAdder item support (optional)
- Geyser/Floodgate cross-play compatibility (optional)
- DragonlordBlessing sell-boost integration (optional)
- CoinsEngine support (optional)

**Admin Controls**
- Full in-game admin GUI for categories, items, and specials
- Reload, pricing resets, and category management
- Permission-based limits and bypass rules
